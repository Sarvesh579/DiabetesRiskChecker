# DiabetesRiskChecker
A webapp that analyses one's risk of acquiring diabetes based on their answers to a questionnaire

This is a webapp that runs on React Vite and Node.js<br>
Additionally, it uses python libraries in the backend<br>
*python libs: sys, os, json, joblib, pandas, math*

This project was made as a part of Data Warehousing and Mining subject practicals in college.
It showcases a *logistic regression model* made on a diabetic patients' dataaset.

* Clone repository
* cd to root of project
* install dependencies
```
npm install
```
* After installation of dependencies, in the **backend/** directory, run
```
node server.js
```
* In the **health-risk-app/** directory, run
```
npm run dev
```
* This should give you a localhost url. 
Copy and Paste this URL in your browser and you land on the web-app!
